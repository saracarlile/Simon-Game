Simo n Game

This Simon Game is a memory game.

Follow the computer, and press the buttons to copy the sequence it presents.

A count will be kept of the number of sequences you copy. You win if your count reaches 20.

Next >>>

You can play using Easy Mode or Strict Mode.

In strict mode, if you press a wrong button in the sequence the game restarts and your count is set to zero.

In easy mode, the count is not reset if you get part of the sequence wrong, and you are able to try again.
